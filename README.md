# Multi Auto Role Extension Documentation

## Overview
The Multi Auto Role extension automatically assigns multiple roles to new members when they join your Discord server. It features a simple interactive interface with minimal commands and maximum functionality.

## Main Command
- `!multirole` - Opens the interactive Multi Auto Role control panel

## Features

### ðŸŽ­ **Multiple Role Assignment**
- Assign multiple roles instantly when users join
- No delays or complex conditions - just simple, fast role assignment
- Automatically skips bots

### ðŸ”„ **Interactive Control Panel**
- **Enable/Disable Button** - Toggle the system on/off with one click
- **Set Roles Button** - Configure which roles to assign by mentioning them
- **Refresh Button** - Update the status display in real-time

### ðŸ“Š **Real-Time Status**
- Shows current system status (Enabled/Disabled)
- Displays number of configured roles
- Lists all roles that will be assigned
- Color-coded embed (Green = Enabled, Red = Disabled)

### âš¡ **Simple Setup Process**
1. Run `!multirole` command
2. Click "Set Roles" button
3. Mention the roles you want: `@Member @Newcomer @Verified`
4. Click "Enable" to activate the system
5. Done! New members will automatically get those roles

## Permissions Required
- **Administrator** permission to use the command
- **Manage Roles** permission for the bot to assign roles
- Bot's role must be higher than the roles it needs to assign

## Key Benefits
- **Zero configuration files** - everything managed through Discord interface
- **Instant setup** - takes less than 30 seconds to configure
- **Visual feedback** - always know the current status at a glance
- **Error handling** - gracefully handles deleted roles and permission issues
- **Lightweight** - minimal resource usage and fast performance
